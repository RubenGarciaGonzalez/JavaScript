¿ Es posible cambiar un objeto declarado como “const” ? Razona tu respuesta: const usuario = { nombre: "Juan" }; // Esto funciona? usuario.nombre = "Diego"; La declaracion de constante crea una referencia de sólo lectura; esto no significa que el valor que tenga es inmutable, sino que no puede ser reasignado, por consiguiente, el objeto si puede ser alterado.
